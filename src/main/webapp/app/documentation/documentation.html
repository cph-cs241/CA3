DESCRIPTION OF THE PROJECT


Usage of angular filters




Fetching data from google places AIP

Since we are not able to make google AIP requests directly from the client we have decided to make the request from the client to our server, from over server to google, then send the result back to the client. It is not the most optimal way of doing it, because we load more traffic onto our server and there is even more steps the information have to travel through to get to the end user, which also makes the application seeming slow.
